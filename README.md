# CryptoNews
### Continuing development on [CryptoTracker All-in-One App](https://github.com/mrmendoza171/cryptotracker)
#### Track your cryptocurrency holdings and view various statistics in an easy to use dashboard.
#### Inspired by and modeled after [Decrypt Media](https://decrypt.co/).

## Features
- Render data from [CryptoNews API](https://cryptonews-api.com/) (Not using active subscription)
- Overview of news articles and links to sites
- Articles categorized by tickers, topics, and sentiment
- YouTube video player for articles that include media
- Add articles to bookmarked list
